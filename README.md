# Online Shopping

##### A website that allows users to create a list of items, a list of cart items. The final shipping and total costs are calculated. (9/21/15)

#### By Logan Wu & Ian McKenney

## Description

Each ```item``` object has the properties ```name``` and ```price``` properties. The shipping cost is calculated based on the difference in zip code between two locations. Users are allowed to edit, add and delete the list of items and add and delete the list of cart items.

## Setup

* Clone the project using the link provided on Github in the Terminal.
* Open ```index.html``` from the main project folder.

## Technologies Used

HTML, CSS, Bootstrap, JavaScript, AngularJS
